# My Expo App

This is a simple Expo application that serves as a starting point for building mobile applications using React Native and TypeScript.

## Project Structure

```
my-expo-app
├── src
│   ├── App.tsx               # Main entry point of the application
│   ├── components            # Reusable components
│   │   └── index.tsx        # Exports for reusable components
│   ├── screens               # Application screens
│   │   └── HomeScreen.tsx    # Main screen of the application
│   └── types                 # TypeScript types and interfaces
│       └── index.ts          # Exports for types and interfaces
├── app.json                  # Expo application configuration
├── package.json              # npm configuration file
├── tsconfig.json             # TypeScript configuration file
└── README.md                 # Project documentation
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd my-expo-app
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npx expo start
   ```

## Features

- TypeScript support for type safety
- Reusable components for better code organization
- Basic navigation setup
